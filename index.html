Trabajo Final SO (Windows)


1.Características

  • Interfaz de Usuario: Windows ofrece una interfaz gráfica de usuario intuitiva y familiar para la mayoría de los
    usuarios, con características como el menú de inicio, la barra de tareas y los escritorios virtuales.

  • Compatibilidad: Altamente compatible con una gran variedad de hardware y software.

  • Soporte Multitarea: Permite ejecutar múltiples aplicaciones y programas simultáneamente.

  • Actualizaciones Regulares: Microsoft proporciona actualizaciones periódicas para mejorar la funcionalidad y seguridad del sistema.

  • Integración: Buena integración con otros servicios de Microsoft como Office, OneDrive y servicios en la nube de Azure.

2.Ventajas y Desventajas

Ventajas:

  • Compatibilidad Amplia: Soporta una gran cantidad de aplicaciones y hardware.

  • Facilidad de Uso: Interfaz intuitiva y familiar.

  • Actualizaciones Frecuentes: Mejoras continuas en seguridad y funcionalidad.

  • Ecosistema Rico: Gran variedad de aplicaciones y servicios disponibles.

Desventajas:

  • Costo: Las licencias pueden ser costosas comparadas con sistemas operativos gratuitos.

  • Vulnerabilidad: Objetivo frecuente de malware y virus debido a su popularidad.

  • Requerimientos de Hardware: Puede necesitar hardware relativamente potente para ejecutar las versiones más recientes de manera eficiente.

3.Estabilidad y Rendimiento

  • Estabilidad: Generalmente, Windows es estable, pero la estabilidad puede variar dependiendo del hardware y la cantidad de software instalado.
    Las versiones recientes como Windows 10 y Windows 11 han mejorado significativamente en términos de estabilidad.

  • Rendimiento: El rendimiento de Windows puede ser excelente en hardware moderno, pero puede degradarse con el tiempo debido a la acumulación de software y archivos.
    Es crucial mantener el sistema optimizado mediante herramientas de limpieza y mantenimiento.

4.Nivel de Seguridad

  • Actualizaciones de Seguridad: Microsoft lanza actualizaciones de seguridad regularmente para proteger contra las últimas amenazas.

  • Software Antimalware: Windows Defender es un software de seguridad integrado que ofrece protección básica contra malware.

  • Firewall: Incluye un firewall incorporado para ayudar a proteger el sistema de accesos no autorizados.

  • Vulnerabilidades: A pesar de las medidas de seguridad, sigue siendo un objetivo frecuente de ataques debido a su gran base de usuarios.
    Los usuarios deben estar atentos a las prácticas de seguridad, como el uso de contraseñas fuertes y el no descargar software de fuentes no confiables.

5.Costo de Licencia

  • Windows 10 Home: Aproximadamente $139 USD.

  • Windows 10 Pro: Aproximadamente $199.99 USD.

  • Windows 11: Similar al costo de Windows 10, aunque los precios pueden variar ligeramente dependiendo del país y las promociones disponibles.
